# Bibliotheca Oratorii Sacratissimorum Cordium

[![CI](https://github.com/apnovoa/oratory-hearts-library/actions/workflows/ci.yml/badge.svg)](https://github.com/apnovoa/oratory-hearts-library/actions/workflows/ci.yml)

A private digital lending library for the Oratory of the Most Sacred Hearts, implementing strict one-copy-one-loan controlled digital lending with automated checkout, PDF watermarking, expiration, and email notifications.

Built with Flask, SQLite, and server-side rendering.
